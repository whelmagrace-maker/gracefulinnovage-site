# Project Title: Multi-Page Digital Service & Booking Web MVP

### Project Description:
Built a fully responsive, multi-page web application designed as a scalable MVP for digital service providers. The goal of this project was to establish a seamless user journey—from discovering services to booking a live consultation and browsing digital products.

### Key Deliverables:
* Structured a cohesive multi-page architecture (index, about, services, portfolio, booking, shop, and contact).
* Designed a dedicated client onboarding/booking layout to minimize friction and improve conversions.
* Implemented clean, semantic folder and code architecture managed entirely through version control (GitHub).
